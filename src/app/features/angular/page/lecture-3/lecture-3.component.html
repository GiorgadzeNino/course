Module 3: Angular Directives

1. рЃарЃљ рЃљрЃарЃўрЃА Directive Angular-рЃерЃў

Directive рЃљрЃарЃўрЃА рЃЏрЃћрЃЦрЃљрЃюрЃўрЃќрЃЏрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃфрЃЋрЃџрЃўрЃА HTML рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃА рЃЦрЃфрЃћрЃЋрЃљрЃА рЃљрЃю рЃњрЃљрЃарЃћрЃњрЃюрЃЮрЃЉрЃљрЃА.

рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃўрЃф рЃАрЃўрЃюрЃљрЃЏрЃЊрЃЋрЃўрЃџрЃћрЃерЃў рЃљрЃарЃўрЃА рЃЊрЃўрЃарЃћрЃЦрЃбрЃўрЃЋрЃљ, рЃЏрЃљрЃњрЃарЃљрЃЏ:

  • рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃА рЃљрЃЦрЃЋрЃА рЃерЃљрЃЉрЃџрЃЮрЃюрЃў (template)
  • рЃЊрЃўрЃарЃћрЃЦрЃбрЃўрЃЋрЃљрЃА Рђћ рЃљрЃарЃљ

рЃЏрЃљрЃарЃбрЃўрЃЋрЃљрЃЊ: рЃЊрЃўрЃарЃћрЃЦрЃбрЃўрЃЋрЃљ = рЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ HTML-рЃўрЃА рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃќрЃћ

2. Directives-рЃўрЃА рЃбрЃўрЃърЃћрЃЉрЃў

Angular-рЃерЃў рЃњрЃЋрЃљрЃЦрЃЋрЃА 3 рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃбрЃўрЃърЃў:

  • Component Directives РђЊ UI рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃћрЃЉрЃў (Template + Class + Styles)
  • Structural Directives РђЊ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ/рЃгрЃљрЃерЃџрЃљ DOM-рЃерЃў (*ngIf, *ngFor)
  • Attribute Directives РђЊ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃА рЃљрЃбрЃарЃўрЃЉрЃБрЃбрЃћрЃЉрЃўрЃА, рЃАрЃбрЃўрЃџрЃўрЃА рЃљрЃю рЃЦрЃфрЃћрЃЋрЃўрЃА рЃерЃћрЃфрЃЋрЃџрЃљ ([ngClass], [ngStyle])

3. Structural Directives (рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃБрЃџрЃў рЃЊрЃўрЃарЃћрЃЦрЃбрЃўрЃЋрЃћрЃЉрЃў)

Structural directive:

  • рЃфрЃЋрЃџрЃўрЃА DOM-рЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљрЃА
  • рЃљрЃЏрЃљрЃбрЃћрЃЉрЃА рЃљрЃю рЃерЃџрЃўрЃА рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃА

рЃерЃћрЃюрЃўрЃерЃЋрЃюрЃљ: рЃЦрЃЋрЃћрЃЏрЃЮрЃЌ рЃЏрЃЮрЃфрЃћрЃЏрЃБрЃџрЃў рЃњрЃЋрЃљрЃЦрЃЋрЃА рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃў рЃЮрЃарЃў рЃњрЃљрЃюрЃАрЃ«рЃЋрЃљрЃЋрЃћрЃЉрЃБрЃџрЃў рЃЋрЃћрЃарЃАрЃўрЃўрЃЌ рЃарЃЮрЃњрЃЮрЃа рЃўрЃгрЃћрЃарЃћрЃЉрЃЮрЃЊрЃљ рЃФрЃЋрЃћрЃџ рЃЋрЃћрЃарЃАрЃўрЃћрЃЉрЃерЃў рЃЊрЃљ рЃарЃЮрЃњрЃЮрЃа рЃљрЃ«рЃљрЃџрЃерЃў. рЃЌрЃБрЃЏрЃфрЃљ рЃБрЃюрЃЊрЃљ рЃљрЃдрЃЋрЃюрЃўрЃерЃюрЃЮрЃЌ рЃарЃЮрЃЏ рЃЌрЃБ рЃљрЃ«рЃљрЃџ рЃЋрЃћрЃарЃАрЃўрЃљрЃерЃўрЃф рЃњрЃЋрЃћрЃћрЃгрЃћрЃарЃћрЃЉрЃљ рЃФрЃЋрЃћрЃџрЃў рЃАрЃўрЃюрЃбрЃљрЃЦрЃАрЃўрЃЌ, рЃЎрЃЮрЃЊрЃў рЃўрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃА рЃЕрЃЋрЃћрЃБрЃџрЃћрЃЉрЃарЃўрЃЋрЃљрЃЊ рЃЊрЃљ рЃљрЃа рЃћрЃЦрЃюрЃћрЃЉрЃљ рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃљ

­ЪћЉ рЃДрЃЮрЃЋрЃћрЃџрЃЌрЃЋрЃўрЃА рЃўрЃгрЃДрЃћрЃЉрЃљ *-рЃўрЃЌ

3.1 *ngIf

<p *ngIf="isLoggedIn">Welcome!</p>
  • рЃћ.рЃў. рЃњрЃЋрЃљрЃЦрЃЋрЃА рЃфрЃЋрЃџрЃљрЃЊрЃў isLoggedIn рЃЊрЃљ рЃЌрЃБ isLoggedIn === true Рєњ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў рЃЕрЃљрЃюрЃА
  • рЃЌрЃБ рЃЏрЃљрЃА рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ рЃљрЃа рЃљрЃЦрЃЋрЃА рЃљрЃю рЃЏрЃўрЃАрЃў рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљрЃљfalse Рєњ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў рЃАрЃљрЃћрЃарЃЌрЃЮрЃЊ рЃљрЃа рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА DOM-рЃерЃў

3.2 *ngFor

<li *ngFor="let item of items">
    {{ item }}
</li>

рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃАрЃўрЃћрЃЉрЃўрЃА рЃЊрЃљрЃАрЃљрЃарЃћрЃюрЃЊрЃћрЃарЃћрЃЉрЃџрЃљрЃЊ. рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў рЃЌрЃљрЃЋрЃўрЃАрЃў рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃўрЃЌ рЃћрЃарЃЌрЃўрЃљрЃюрЃљрЃЊ рЃњрЃљрЃЏрЃЮрЃЊрЃўрЃА DOM-рЃерЃў рЃарЃЮрЃњрЃЮрЃарЃф <li> рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў.

рЃарЃЮрЃЊрЃћрЃАрЃљрЃф *ngFor-рЃА рЃљрЃа рЃБрЃгрЃћрЃарЃўрЃљ trackBy рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ, рЃарЃЮрЃЊрЃћрЃАрЃљрЃф рЃџрЃўрЃАрЃбрЃў рЃњрЃљрЃюрЃљрЃ«рЃџрЃЊрЃћрЃЉрЃљ, рЃљрЃюрЃњрЃБрЃџрЃљрЃарЃў рЃЊрЃЮрЃЏрЃўрЃЊрЃљрЃю рЃгрЃљрЃерЃџрЃўрЃА рЃДрЃЋрЃћрЃџрЃљ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃА рЃЊрЃљ рЃљрЃ«рЃџрЃўрЃЊрЃљрЃю рЃЊрЃљрЃ«рЃљрЃбрЃљрЃЋрЃА, рЃарЃЮрЃЊрЃћрЃАрЃљрЃф рЃЉрЃћрЃЋрЃарЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃў рЃњрЃЋрЃљрЃЦрЃЋрЃА, рЃЊрЃљрЃЏрЃћрЃЌрЃљрЃюрЃ«рЃЏрЃћрЃЉрЃўрЃЌ, рЃърЃћрЃарЃцрЃЮрЃарЃЏрЃљрЃАрЃўрЃАрЃЌрЃЋрЃўрЃА рЃфрЃБрЃЊрЃўрЃљ рЃћрЃарЃЌрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃўрЃА рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮ рЃДрЃЋрЃћрЃџрЃљрЃА рЃЌрЃљрЃЋрЃўрЃЊрЃљрЃю рЃњрЃљрЃЊрЃљрЃгрЃћрЃарЃљ

trackBy рЃцрЃБрЃюрЃЦрЃфрЃўрЃўрЃЌ рЃЎрЃў рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃА рЃЋрЃљрЃюрЃўрЃГрЃћрЃЉрЃЌ рЃарЃљрЃдрЃљрЃф рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃа рЃЏрЃюрЃўрЃерЃюрЃћрЃџрЃЮрЃЉрЃљрЃА, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃБрЃерЃБрЃљрЃџрЃЮрЃЊ рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃБрЃџ рЃљрЃю рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃБрЃџ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃА рЃерЃћрЃфрЃЋрЃџрЃўрЃА рЃЊрЃЮрЃЏрЃерЃў рЃЊрЃљ рЃљрЃа рЃњрЃљрЃЊрЃљрЃ«рЃљрЃбрЃљрЃЋрЃА рЃљрЃ«рЃџрЃўрЃЊрЃљрЃю рЃЏрЃЌрЃџрЃўрЃљрЃю рЃџрЃўрЃАрЃбрЃА

3.3 *ngSwitch

<div [ngSwitch]="role">
    <p *ngSwitchCase="'admin'">Admin</p>
    <p *ngSwitchCase="'user'">User</p>
    <p *ngSwitchDefault>Guest</p>
    </div>

Рё╣№ИЈ рЃљрЃ«рЃљрЃџрЃў рЃАрЃўрЃюрЃбрЃљрЃЦрЃАрЃў: Angular-рЃўрЃА рЃЉрЃЮрЃџрЃЮ рЃЋрЃћрЃарЃАрЃўрЃћрЃЉрЃерЃў рЃерЃћрЃњрЃЋрЃўрЃФрЃџрЃўрЃљ template-рЃерЃў рЃњрЃљрЃЏрЃЮрЃЋрЃўрЃДрЃћрЃюрЃЮрЃЌ @if, @for рЃЊрЃљ @switch рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃБрЃџрЃў рЃЊрЃўрЃарЃћрЃЦрЃбрЃўрЃЋрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА.

­ЪћЉ 3.1 @if

@if (isLoggedIn) {
  <p>Welcome!</p>
} @else {
  <p>Please log in</p>
}
  • isLoggedIn рЃљрЃарЃўрЃА рЃфрЃЋрЃџрЃљрЃЊрЃў.
  • рЃЌрЃБ isLoggedIn === true Рєњ DOM-рЃерЃў рЃЕрЃљрЃюрЃА <p>Welcome!</p>.
  • рЃЌрЃБ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ рЃљрЃарЃўрЃА false рЃљрЃю undefined Рєњ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА @else рЃЉрЃџрЃЮрЃЎрЃў рЃЊрЃљ рЃњрЃљрЃћрЃерЃЋрЃћрЃЉрЃљ <p>Please log in</p>.

­ЪћЉ 3.2 @for

<ul>
  @for (item of items; track item) {
    <li>{{ item }}</li>
  }
</ul>
  • items рЃљрЃарЃўрЃА рЃАрЃўрЃљ/рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЏрЃљрЃАрЃўрЃЋрЃў (рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ: ['Angular', 'React', 'Vue']).
  • рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў рЃЌрЃљрЃЋрЃўрЃАрЃў рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃўрЃЌ рЃћрЃарЃЌрЃўрЃљрЃюрЃљрЃЊ рЃњрЃљрЃЏрЃЮрЃЊрЃўрЃА DOM-рЃерЃў рЃарЃЮрЃњрЃЮрЃарЃф <li> рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў.
  • track item рЃћрЃ«рЃЏрЃљрЃарЃћрЃЉрЃљ Angular-рЃА рЃћрЃцрЃћрЃЦрЃбрЃБрЃарЃљрЃЊ рЃњрЃљрЃюрЃљрЃљрЃ«рЃџрЃЮрЃА рЃАрЃўрЃљ.

­ЪћЉ 3.3 @switch

@switch (role) {
  @case ('admin') { <p>Admin</p> }
  @case ('user')  { <p>User</p> }
  @default        { <p>Guest</p> }
}
  • role рЃљрЃарЃўрЃА рЃфрЃЋрЃџрЃљрЃЊрЃў (рЃЏрЃљрЃњ. 'admin', 'user' рЃљрЃю 'guest').
  • DOM-рЃерЃў рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃўрЃА <p> рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў рЃЕрЃюрЃЊрЃћрЃЉрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљ рЃЏрЃўрЃЏрЃЊрЃўрЃюрЃљрЃарЃћ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљрЃА.
  • рЃЌрЃБ рЃљрЃарЃф рЃћрЃарЃЌрЃў @case рЃљрЃа рЃЊрЃљрЃћрЃЏрЃЌрЃ«рЃЋрЃљ Рєњ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА @default рЃЉрЃџрЃЮрЃЎрЃў.

trackBy vs track - рЃерЃћрЃЊрЃљрЃарЃћрЃЉрЃљ

рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃЏрЃўрЃќрЃљрЃюрЃў рЃАрЃљрЃЋрЃљрЃџрЃЊрЃћрЃЉрЃБрЃџрЃЮ?
